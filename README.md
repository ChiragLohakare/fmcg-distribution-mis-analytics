Operational MIS & Analytics System for FMCG Distribution
Excel | PostgreSQL | Power BI

About This Project
I built this project to simulate how a real FMCG distribution company tracks its sales, inventory, logistics, and costs. The company used in this project is Shreenath Agro & FMCG Distributors Pvt. Ltd., based in Nagpur, Maharashtra — covering the Vidarbha region for FY 2024.
The idea was to build something end-to-end — starting from raw data all the way to an automated MIS report and an executive dashboard — the kind of work a Data or MIS Analyst would actually do on the job.

What I Built

Designed a dataset with 9,400+ sales transactions, 30 SKUs, 35 customers, and 8 sales regions
Created a full PostgreSQL database with 10 tables and wrote KPI queries for sales, inventory, and logistics
Built a 3-page Power BI dashboard for management reporting
Made an Excel MIS report with formulas and a separate macro-automated version for weekly reporting
